Features 
Basically this page describes the reasons why to adopt a dog.There are various reasons described for which users adopt a dog. On this page, we havve two images and when user hover over the image i have added an overlay element which adds animation. To add overlay element i have taken help from w3schools.https://www.w3schools.com/howto/howto_css_image_overlay_title.asp

Login Page
In login page user has to enter his email and password and then only submit button will be enabled. Until then, it will be disabled. I have included all the validations which are if user does not provide the email address, the input box will be highlighted in red and user will receive the error message that “Email is required”. Similarly, if user does not provide password an error message will be displayed that “Password is required”. Also, if email is not in proper format that is abc@example.com , an error message will be displayed “Email should be a valid  address”.
Forgot Password page
User will be navigated to this page on clicking Forgot password page on Login page -Dheeraj. On this page I have included email input box which will take user email and once user clicks on Submit button a message will be displayed that “A activation link has been set on your email.”
Pet Listing Page
User will be able to view the pets which are available to adopt. It will have list of pets and on clicking on More Info user will be navigated to the pet profile page.



Product listing page
User will be able to view the pets which are available to adopt. It will have list of pets and on clicking on More Info user will be navigated to the pet profile page.
